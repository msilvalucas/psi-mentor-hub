

# Landing Page — PsiPraQueTeQuero 🧠

Landing page moderna para o clube de desenvolvimento continuado para psicólogos clínicos, usando a paleta de cores marrom (#6f4327) e branco (#fff).

---

## Seções da Página

### 1. Header / Navegação
- Logo "PsiPraQueTeQuero" à esquerda
- Menu de navegação: Sobre Nós, Serviços, Contato
- Ícones das redes sociais (WhatsApp, Instagram, TikTok, YouTube) — links placeholder por enquanto, para serem substituídos pelos reais

### 2. Hero — "Bem-vindo!"
- Título grande de boas-vindas com texto de impacto sobre o clube
- Espaço para foto das mentoras (placeholder por enquanto)
- Botão CTA "Quero saber mais" direcionando para a seção de serviços

### 3. Sobre Nós
- Texto sobre as fundadoras: +7 mil horas de atendimento, desafios da clínica, trajetória de formação
- Destaque para a missão: espaço de apoio para prática clínica ética, segura e autoral
- Layout em duas colunas: texto + imagem placeholder das mentoras

### 4. Serviços — "Competências Essenciais na Prática Clínica"
- Card destacando o módulo principal com descrição completa
- Seção de identificação com o público-alvo: "Eu me formei e quero viver da clínica, mas não sei por onde começar..."
- Cards com os principais pontos de dor: insegurança, dúvidas técnicas, impasses éticos

### 5. Formulário de Contato
- Campos: Nome, Email, Mensagem
- Validação dos campos com feedback visual
- Dados salvos localmente (sem backend por enquanto)

### 6. Botão Flutuante de WhatsApp
- Botão fixo no canto inferior direito da tela
- Link para WhatsApp (placeholder, para ser substituído pelo número real)

### 7. Footer
- Logo, links das redes sociais, copyright

---

## Design & Estilo
- **Paleta**: Marrom (#6f4327) como cor primária, branco (#fff) como fundo, tons quentes de apoio
- **Tipografia**: Elegante e profissional, com hierarquia clara
- **Animações**: Scroll suave, fade-in nas seções ao rolar a página
- **Responsivo**: Adaptado para mobile e desktop
